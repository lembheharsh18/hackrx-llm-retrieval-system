# HackRx LLM-Powered Query Retrieval System

An intelligent document analysis system for insurance, legal, HR, and compliance documents using Gemini AI and FAISS.

## Features
- PDF/DOCX document processing
- Semantic search with FAISS
- Gemini AI integration
- Real-time query processing

## Quick Start
1. Install dependencies: `pip install -r requirements.txt`
2. Set up .env file with your API keys
3. Run: `uvicorn src.api.main:app --reload`
